# Übung 10: Insertion Sort

Implementiere den Sortieralgorithmus **Insertion Sort** für ein Array von Ganzzahlen.

- Gehe dabei schrittweise vor und orientiere dich am im Arbeitsblatt beschriebenen Ablauf.
- Verwende keine vorgefertigten Sortierfunktionen.
- Der Algorithmus soll das Array **aufsteigend** sortieren.
- Achte besonders auf die korrekte Verwendung der Indizes und den Kontrollfluss der Schleifen.

Ergänze deinen Code optional mit Ausgaben oder Debugging-Schritten, um den Ablauf während der Sortierung nachvollziehen zu können.

# Übung 11: Selection Sort

Implementiere den Sortieralgorithmus **Selection Sort** für ein Array von Zeichenketten (`string[]`).

- Die Strings sollen **alphabetisch aufsteigend** sortiert werden.
- Verwende zum Vergleichen der Strings eine geeignete Vergleichsoperation.
- Verzichte auf LINQ oder Bibliotheksfunktionen zum Sortieren.
- Der Algorithmus soll das kleinste Element im unsortierten Bereich suchen und an die richtige Position tauschen.

Überlege dir nach der Implementierung, wie sich der Ablauf von Selection Sort vom Insertion Sort unterscheidet.
